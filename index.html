DheKasir Platinum

Version 22.6 STABLE

Solusi kasir retail modern terpadu untuk UMKM Indonesia. Membantu Anda mengelola transaksi, stok, dan piutang pelanggan dengan mudah dan aman secara Cloud.

Privasi & Kebijakan

1. Pengumpulan Data: DheKasir Platinum menyimpan data transaksi dan inventaris Anda secara lokal di perangkat Anda (LocalStorage) dan secara opsional di Google Spreadsheet pribadi Anda melalui integrasi URL API.

2. Keamanan: Kami tidak memiliki akses ke database Google Spreadsheet Anda. Keamanan data sepenuhnya bergantung pada kerahasiaan URL Google Apps Script yang Anda buat.

3. Penggunaan Kamera: Izin kamera hanya digunakan untuk fitur pemindaian barcode produk dan tidak menyimpan gambar atau video apapun ke server luar.

4. Bluetooth: Fitur Bluetooth hanya digunakan untuk berkomunikasi dengan printer thermal eksternal saat proses pencetakan struk.

5. Perubahan: Kami berhak memperbarui kebijakan ini sewaktu-waktu untuk meningkatkan layanan aplikasi.
